# AI Traffic Analyser

Computer vision project that analyses footage of a local roundabout/intersection and produces traffic analytics: vehicle counts by type, most common routes through the junction, and traffic patterns over time.

## Planned pipeline
1. Vehicle detection & classification (per frame)
2. Multi-object tracking (persistent identity across frames)
3. Route extraction via entry/exit zone crossings
4. Analytics aggregation (counts by class/route/time)
5. Reporting (plots / dashboard)

## Status
Early setup — detection pipeline not yet implemented.
